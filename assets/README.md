# Assets Directory

This directory contains placeholder assets for the Expo app.

## Current Files

- `icon.png` - App icon (copied from Android xxxhdpi launcher icon)
- `adaptive-icon.png` - Android adaptive icon (copied from Android xxxhdpi launcher icon)
- `splash.png` - Splash screen (placeholder, same as icon)

## Required Asset Specifications

### icon.png
- Size: 1024x1024 pixels
- Format: PNG
- Purpose: App icon for both iOS and Android

### adaptive-icon.png
- Size: 1024x1024 pixels
- Format: PNG
- Purpose: Android adaptive icon foreground

### splash.png
- Size: 1284x2778 pixels (or similar high resolution)
- Format: PNG
- Purpose: Splash screen shown while app loads
